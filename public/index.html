Disk browser is an open-source file manager released under MIT license. This package has been created using test driven approach and a full test suite is available.

Git Repositories -

  • For Client side source code -
    https://github.com/ershwetabansal/disk-browser
  • For Server side Laravel PHP source code -
    https://github.com/ershwetabansal/disk-browser-laravel
  • For Server side Node JS source code -
    https://github.com/ershwetabansal/disk-browser-nodejs
  • Can be integrated with any server side languages by exposing RESTFUL based APIs.
  • Support is available for LARAVEL PHP and node JS. Can be very easily extended in any other language.
  • Disk browser client library is written using node and compiled using browserify.
  • Test cases are written using Jasmine and Karma is used as test runner.
  • Developed on Jquery and bootstrap frameworks.
  • Customizable and supports multiple disks in one instance.
  • Displayed as modal box.
  • Easy integration with editors like CKEditor, TinyMCE. Current support is available for TinyMCE.
  • Easy integration with AWS S3 storage or any other file storage server. Support is available in Laravel PHP.
  • Ability to upload, delete, rename, download files. Can be used to only fetch File path.
  • Ability to create and rename folders.
  • Session based authentication.
  • Opening exclusively a given folder as disk.
  • Allows to apply file extension restriction.
  • Multiple disks configuration is allowed.
  • Allows to configure read only disks.
  • Multiple disks search capability.
  • Very easy to use.
  • Switch from list to grid view and vice-versa.
